# Feedback Form (Vue 3)

## 📌 Описание

Это одностраничное приложение (SPA), реализованное на Vue 3.
Приложение представляет собой форму обратной связи.

---

### 1. Клонировать репозиторий

```bash
git clone https://github.com/Scat55/feedback-form.git
cd feedback-form
```

---

## 🏗 Структура проекта

```
.
├── src/            # исходный код приложения
├── public/         # статические файлы
├── dist/           # production-сборка
```

---

## 🔧 Локальная разработка

```bash
npm install
npm run dev
```

---

## ⚙️ Сборка проекта

```bash
npm run build
```

Результат сборки находится в папке `dist/`.
